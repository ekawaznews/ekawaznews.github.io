Cloud Storage Manager

Connect multiple Google Drive accounts or other cloud providers. Upload images, videos, PDFs, audio with unlimited storage via your own cloud accounts.

ℹ️ How Cloud Storage Works Upload files to your Google Drive or cloud service, then paste the public share link here. Your files are served directly from your cloud account. No size limit from this website. You can add multiple accounts for extra capacity.

➕ Add Cloud Account

🔥 Firebase Setup Steps:
1. Go to firebase.google.com → Create a new project
2. Click ⚙️ → Project Settings → scroll to "Your Apps"
3. Click </> Web icon → Register app → copy the config values
4. Go to Firestore Database → Create database → Start in Test Mode
5. Paste each value below. Free plan gives 1GB text storage
After adding, you can set this as the Active Firebase from the accounts list below. All posts and settings will sync to the active Firebase project.
📋 Google Drive Setup Steps:
1. Go to drive.google.com and create a folder for your media
2. Right-click folder → Share → Set to "Anyone with the link can view"
3. Copy the Folder ID from the URL: drive.google.com/drive/folders/[FOLDER-ID]
4. Paste the Folder ID below. Files shared from this folder will be accessible.
📋 Cloudinary Setup:
1. Sign up at cloudinary.com (free plan: 25GB/month)
2. Go to Dashboard → copy your Cloud Name and Upload Preset
3. Create an unsigned upload preset in Settings → Upload → Add upload preset
📋 ImgBB Setup:
1. Sign up at imgbb.com → go to API tab
2. Generate an API key and paste it below
📋 Dropbox Setup:
Upload files to Dropbox → Right-click → Share → Copy link. Change ?dl=0 to ?raw=1 in the URL for direct access. Paste file URLs when adding media.
📋 OneDrive Setup:
Upload files to OneDrive → Right-click → Embed → Copy the embed src URL. Use those URLs when adding media to articles.
